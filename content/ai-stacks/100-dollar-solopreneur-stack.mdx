## When to Add a Fifth Tool

Add a fifth tool only when one of these breaks:
- **Hitting Jasper's word cap?** Upgrade plan, don't add a second writer.
- **Need video?** Add Descript ($24/mo).
- **Need scheduling?** Add Buffer or Hypefury (~$15/mo).

Don't add tools you're not actively using daily. Tool sprawl kills solopreneurs.
