## How These Four Tools Work Together

The stack assumes a **layered model of agency**:

1. **Copilot** — sub-second, single-line completions. No agency.
2. **Cursor** — minute-scale, file-level edits. Bounded agency.
3. **Claude Code** — task-level, repo-wide changes. Full agency.
4. **Sentry** — production observability. Catches what slipped through.

Each tool sits at a different latency / risk tier. You don't use Claude Code to autocomplete a variable name; you don't use Copilot to refactor a service.

## What I Don't Run in This Stack (and Why)

- **A second chat IDE.** Pick Cursor or Windsurf. Running both is tool sprawl.
- **A local coding model.** Opus 4.7 task budgets killed my main reason (predictable cost) for running local Llama. The local option is great for privacy, not for performance.
- **AI-driven testing tools.** Sentry catches what matters in production. Adding an AI test-gen tool added flakes more than it caught regressions in our tests.

## When to Drop or Add a Tool

**Drop GitHub Copilot if:** You ship 100% via Cursor's tab model. Cursor's tab is competitive in 2026, so this cuts the stack to **$66/mo**.

**Drop Cursor if:** You go all-in on Claude Code's CLI workflow. That cuts the stack to **$56/mo**, but you lose the polished IDE.

**Add Linear ($8/seat) if:** Your team needs ticket-to-PR automation. Claude Code now ships with native Linear integration.

**Add Devin / OpenHands ($25/mo+):** For fully autonomous "give it a Linear ticket, get a PR" workflows. Still not ready for prime time in our testing, but watch this space.

## Frequently Asked Questions

**Why not just use one tool?**
Each tool sits at a different latency tier. Single-tool stacks either burn tokens on trivial autocomplete (overkill) or fail at agentic refactors (underkill).

**Is $76/mo really enough for a production team?**
Per developer, yes. Team plans scale linearly. The Sentry slot is the only one where team pricing kicks in past 5 seats.

Swap in Claude in Cursor + Aider as a CLI fallback. Same idea, slightly less polished.

Yes — drop Sentry's Team plan for the Developer plan ($0–$8). New total: $50–58/mo.
