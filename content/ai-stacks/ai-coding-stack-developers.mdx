## Why This Stack

These four tools cover the full developer loop: **autocomplete in the editor, agentic refactors across files, UI prototyping, and a general-purpose thinking partner.** They're complementary, not redundant.

---

## The Stack

### 1. Inline Autocomplete → GitHub Copilot ($10/mo)

The cheapest, most reliable inline suggestion engine. Live in any editor.

## What I Don't Use

- **Multiple chat models in the IDE.** Pick one. Switching costs more than it saves.
- **Tab-completion shootouts.** Copilot is fine. Stop tweaking.
- **Local LLMs for daily work.** Great for privacy use cases, slow for daily coding.

---

## When to Drop a Tool

If you're using Cursor heavily, you can probably drop Copilot — Cursor's tab model is competitive. That cuts the stack to **~$40/mo**.
